검사 관련

1. 현재 검사 가능한 학과-학번은 아래과 같습니다. 검사대상에 속하지 않다면 회원가입이 불가능합니다. 꼭 검사대상인지 확인하세요!

- 검사대상 확인 :  검사대상표

2. 교직, 전과생, 편입생, 외국학생에 해당하는 사용자는 검사 기준이 따로 설정되지 않았습니다.

3. 연필/복필 기준은 15학점, 연선/복선 기준은 24학점으로만 설정되어 있습니다. 특수한 경우는 반영되지 않았습니다. (ex. 법학과, 건축학과, 항공시스템학과, 국방시스템공학과 복전)

※ 21년도 이후에 신청한 연계/융합전공 신청자는 졸업기준이 변경되었습니다. 현재 이는 반영되지 않았으므로 주의하시기 바랍니다!

4. 추천 과목 정보는 2020-2, 2021-1학기에 열렸던 최신 기준으로 제공됩니다. (ex. 과거엔 서양철학이 2학점이였다면 현재는 3학점으로 변경됨)

5. 검사 기준은 최신버전 수강편람(2021.02.22)을 반영하여 설정되었으며, 수강편람은 매년 개편되므로 자신이 알고 있는 구버전 수강편람과 다를 수 있습니다.

- 세종대학교 교과과정 참고 링크 :    수강편람    공학인증

6. 세종대학교에서 공식적으로 만든 사이트가 아니므로 검사 결과가 정확하지 않을 수 있습니다. 오류발생 시 상단 메뉴바 링크로 피드백 부탁드립니다.

안내 사항

1. 학생인증을 위해선  세종대학교포털 사이트에서 개인정보수집 동의가 되어있어야합니다. (보통 동의되어있음)

- 로그인 -> 정보수정 -> 개인정보수집동의 모두 동의

2. 학생인증 시  대양휴머니티칼리지 사이트에서 학생 기본 정보와 고전독서의 영역별 인증 현황을 가져옵니다.

- 고전독서인증 : 대양휴머니티칼리지 -> 고전독서인증센터 -> 영역별 인증현황

3. 사용자의 세종대학교 포털 사이트의 비밀번호는 인증에만 1회 사용되며 저장되지 않습니다.

4. 검사를 위해선 기이수성적 엑셀파일을 직접 업로드해야하므로 PC환경에서 진행하는 것을 권장합니다.

5. 사용자가 업로드하는 기이수성적 엑셀파일은 저장되지 않습니다.

6. 사용자가 들었던 과목 정보만이 머신러닝을 위한 용도로 데이터베이스로 변환하여 저장됩니다. (평점은 저장되지 않습니다.)

7. 업로드 시 성적 중 평점이 F, NP인 과목은 인정되지 않고 자동 삭제됩니다.

8. 저장된 사용자 데이터베이스는 과목추천 및 교양과목 통계에 사용됩니다.
