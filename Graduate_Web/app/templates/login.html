검사 관련

1. 현재 검사 가능한 학과-학번은 아래와 같습니다. (아래 표에 해당하지 않으면 검사가 불가합니다.)

학과-학번 데이터사이언스 17~21 디콘,소프트웨어 15~21 정보보호 15~21 지능기전 17~21 컴퓨터공학 15~21
졸업검사 O O O O O
공학인증검사 X 15제외 가능 X X X

2. 현재 전과생, 편입생, 연계전공, 복수전공의 사례는 전공 부분에서 정확한 검사가 불가합니다. (다른 영역은 검사가능)

3. 수강편람은 최신버전(2021)을 참고했으며, 수강편람은 매년 개편되므로 자신이 알고 있는 구버전 수강편람과 다를 수 있습니다.

4. 강의 정보 또한 최신 기준으로 출력됩니다. (ex. 과거엔 서양철학이 2학점이였다면 현재는 3학점으로 변경됨)

5. 세종대학교 교과과정 링크 :    수강편람    공학인증

안내 사항

1. 로그인 시  UIS 사이트와 대양휴머니티칼리지 사이트에서 졸업요건 검사에 필요한 데이터를 자동으로 가져옵니다.

- 고전독서인증 : 대양휴머니티칼리지 -> 고전독서인증센터 -> 영역별 인증현황

- 기이수과목 : uis -> 성적 및 강의평가 -> 기이수성적조회 -> 성적엑셀다운로드

- 영어인증 : uis -> 졸업적부심사 및 자가진단 -> 과목이수점검 -> 영어졸업인증

2. 사용자의 uis/대양휴머니티 사이트의 아이디와 비밀번호는 저장되지 않습니다.

3. 사용자의 기이수성적 엑셀파일은 즉시 삭제됩니다.

4. 사용자가 들었던 과목 정보만이 머신러닝을 위한 용도로 데이터베이스에 저장됩니다. (학점은 저장되지 않습니다.)

5. 검사 시 성적 중 학점이 F, NP인 과목은 인정되지 않고 자동 삭제됩니다.

6. 관리자는 저장된 데이터베이스를 절대 다른 용도로 사용하지 않습니다.

7. PC환경에서 Chrome 브라우저를 사용하는 것을 권장합니다. (모바일도 사용 가능)

8. 검사 결과가 정확하지 않을 수 있습니다. 오류발생 시 피드백 부탁드립니다. (상단 메뉴바 링크)
